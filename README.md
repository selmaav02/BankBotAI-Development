# 🏦 NexaBank — AI-Powered Smart Banking

**NexaBank** is a modern AI-powered banking application that combines a premium web interface with an intelligent virtual assistant to deliver a smart, secure, and interactive banking experience.

---

## ✨ Overview

NexaBank simulates a real-world digital banking system where users can manage accounts, perform transactions, and get instant assistance through an AI chatbot.

Unlike traditional banking apps, NexaBank focuses on **interaction and intelligence**, allowing users to communicate naturally with the system.

---

## 👤 User Features

* Secure login for users and admins
* Dashboard with account details (Savings & Current)
* View recent transactions
* Fund transfers and bill payments
* Loan application system
* Integrated chatbot available on all pages

---

## 🤖 NexaBot (AI Assistant)

NexaBot is an intelligent virtual assistant powered by Rasa.

It can:

* Check account balance
* Provide loan information
* Assist with fund transfers
* Help with KYC updates
* Block cards in emergency
* Locate nearby ATMs and branches

---

## 🛠️ Admin Features

* Monitor chatbot performance
* View user interaction analytics
* Manage FAQ and training data
* Export logs for analysis
* Monitor system health

---

## ⚙️ Tech Stack

* Backend: Python (Flask)
